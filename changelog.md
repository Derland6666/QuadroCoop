# Changelog


## v1.0.0

* **Initial Public Release**
* 
* **Core Engine:** Added 4-player local co-op physics engine.
* 
* **Input:** Added support for independent keyboard bindings (P1-P4) and mobile quadrant touch zones.
* 
* **Trigger System:** Implemented position-delta cloning to synchronize Move, Rotate, and Follow triggers across all
  players.
  
* **Dual Portals:** Added full dual portal support (IDs 99/101) for independent dual-mode toggling for P3 and P4.
* 
* **Settings:** Added in-game Settings Popup via Pause Menu to enable/disable P3 and P4.
* 
* **Visuals:** Unique color schemes and overhead labels for each player (P1=Gold, P2=Blue, P3=Red, P4=Green).
